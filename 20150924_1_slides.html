The Box model

margin / border / padding

定位 / 大小

position / width / height / top / left / right / bottom

The Position

  • static (default): 就是原本的樣子，這時候上下左右是沒用的
  • absolute: 由 上下左右 絕對定位 於 上個有設定定位的元件
  • fixed: 由 上下左右 絕對定位 於 viewport
  • relative: 由 上下左右 偏移原本的位置
